# 👟 Shoes Website - Premium E-Commerce Front-End

Welcome to the **Shoes Website** project! This is a modern, clean, and fully responsive e-commerce front-end interface designed to showcase premium footwear (Sneakers, Boots, etc.) with an interactive user experience.

## 🚀 Live Demo
You can check out the live website here:
👉 **https://haseebdev0.github.io/shoes-website/**

---

## ✨ Features
- **Modern UI/UX:** Clean, dark/light contrast themed design with high-quality shoe visuals.
- **Dynamic Tab Switching:** Smooth navigation between different categories (Sneakers, Boots, etc.) without reloading the page.
- **Interactive Cart System:** Users can add products to the cart with a real-time update on the interface.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile screens.
- **Modular Architecture:** Well-structured code separating HTML, CSS, and JavaScript into professional production-ready directories.
